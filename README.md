# Lexical Analyzer

## Getting Started

Download files and make sure all are in the same directory

### Prerequisites

gcc compiler


## Running the tests

An input file is already given, simply run lex.c with gcc and check output.txt for the tokens. 

To change input, simply change the content inside input.txt. DO NOT change any file names. If a new input file is needed, delete the given input file and change your input file to “input.txt”
